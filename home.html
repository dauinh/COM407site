Evolving a fuzzy Xpilot controller

COM407 final project

View code

Project description

We want to create a fuzzy system to control Xpilot agent, based on the homework earlier this semester. There are two drawbacks in the previous system that we want to tackle this time: (1) the values ranges in the old fuzzy system need to be carefully researched and calculated; (2) the defuzzification process may require some rewriting. In addition, as our old system focuses primarily on avoiding walls, we now want to add a separate set of fuzzy rules that control aiming and shooting of the agent. In addition, during the semester we’re quite intrigued by the concept of genetic algorithms. Therefore, if time permits, we may explore the possibility of combining GA and fuzzy systems, where we use GA to learn the correct value ranges for the fuzzy system.
